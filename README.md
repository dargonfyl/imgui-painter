# IMGUI Painter
A simple imgui painter.

## Requirements
You will need:
- OpenGL version 3.3+ capable system
- CMake 3.0+ installed

## Building
Run the following:
```
$ mkdir build
$ cd build
$ cmake .. [-G your-target-generator]
$ cmake --build .
```
